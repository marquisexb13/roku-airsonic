# Subsonic Roku (Skeleton)

A minimal **Roku SceneGraph** project skeleton that is designed to be wired to a **Subsonic/OpenSubsonic** backend.

## Project layout

Roku SceneGraph packages typically include `manifest`, `source/`, `components/`, and `images/`. The app package root contains the `manifest` file and the directories. (Roku loads all `.xml` under `components/` on launch.)
